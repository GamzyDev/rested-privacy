Rested Privacy Policy

Publisher: Gamzy
Last updated: June 10, 2026

Rested is a Windows desktop application published by Gamzy. This Privacy Policy explains what information Rested may access, how that information is used, where it is stored, and what controls users have.

Information Rested may access

Rested processes data locally on the user's Windows device. To provide its functionality, Rested may read local Codex usage, session, thread, activity, settings, and related application files selected by the user or available on the user's device.

Depending on the user's local files, this data may include usage statistics, session and thread metadata, timestamps, project or file paths, local app settings, activity information, limit information, and text contained in local Codex-related files. Rested uses this information to display usage statistics, session and thread summaries, limits, activity information, and related productivity information inside the app.

How information is used

The information accessed by Rested is used only to provide the app's local functionality. Rested does not sell personal data, does not use personal data for advertising, and does not intentionally transmit Codex usage data, session data, thread data, activity data, local app data, or local settings to Gamzy servers or to third parties.

Storage, retention, and security

Rested stores its own settings and cached app data locally on the user's device. Data shown by the app is derived from local files on the user's device. Rested does not operate a cloud account system for this data.

Because Rested is designed to process this information locally, the security of local files and local app data depends in part on the user's Windows device security, account access, storage permissions, backup settings, and operating system configuration.

Sharing and disclosure

Rested does not intentionally disclose the user's local Codex usage data, session data, thread data, activity data, local app data, or local settings to Gamzy servers, advertisers, data brokers, or other third parties.

If Microsoft Store services are used for purchase, trial, licensing, installation, analytics, crash reporting, certification, app distribution, or app updates, Microsoft may process data according to Microsoft's own privacy terms and platform policies.

User controls

Users control the local files available on their device. Users can stop Rested from displaying information from local Codex files by deleting, moving, or restricting access to those files. If the user deletes or moves the local files that Rested reads, Rested may no longer be able to display information from those files.

Users can remove Rested's local settings and cached app data by clearing the app's local data through Windows settings, removing local app data manually where permitted by Windows, or uninstalling the app.

Sensitive information and children

Rested does not knowingly collect children's personal information, health data, financial data, contacts, photos, microphone recordings, camera recordings, or location data. Rested is not directed to children.

Changes to this policy

Gamzy may update this Privacy Policy when Rested's functionality, data practices, or legal requirements change. The updated policy will be posted on this page with a revised last updated date.
