Rested Privacy Policy

Publisher: Gamzy

Last updated: June 10, 2026

Rested is a Windows desktop application published by Gamzy.

Rested processes data locally on the user's Windows device to display usage, status, session, thread, activity, and limit information.

Rested may read local application files, local Codex usage data, local session data, local thread data, local activity data, and local app settings available on the user's device. This data is used only to provide the app's local functionality.

Rested does not sell personal data, does not use personal data for advertising, and does not intentionally transmit Codex usage data, session data, thread data, activity data, or local app data to Gamzy servers.

Rested stores its own settings and cached app data locally on the user's device. Users can remove this data by clearing the app's local data or uninstalling the app.

If Microsoft Store services are used for purchase, trial, licensing, installation, analytics, crash reporting, or certification, Microsoft may process data according to Microsoft's own privacy terms and platform policies.

Rested does not knowingly collect children's personal information, health data, financial data, contacts, photos, microphone recordings, camera recordings, or location data.
